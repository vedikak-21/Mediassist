# MediAssist

## AI-Powered Healthcare Companion

MediAssist is a Flask-based web application that uses Generative AI to help users understand medical reports, answer health-related questions, and provide personalized health insights.

## Features

- 🤖 AI-powered health assistant using Google Gemini
- 📄 Medical report upload and AI analysis
- 💬 Interactive health chatbot
- 👤 Secure user authentication
- 📊 Health dashboard
- 📝 Chat history
- 📱 Responsive web interface

## Tech Stack

- Python
- Flask
- Google Gemini API
- HTML
- CSS
- JavaScript
- SQLite

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Disclaimer

MediAssist provides AI-generated health information for educational purposes only.

It is **not a substitute for professional medical advice, diagnosis, or treatment.**
